## Тематика сайта, название

Игра "Прыгун". Идея создать игру по типу [Doodle_Jump](https://ru.wikipedia.org/wiki/Doodle_Jump) 

## Цель 

Создать простую и увлекательную игру, направленную на развлечение и расслабление пользователей

## Целевая аудитория

Целевой аудиторией являются подростки (12+), которые активно проводят свободное время в гаджетах, чаще всего в поисках развлекательного контента

## Функциональные возможности

* выбор своего начального прыгуна;
* разблокировка новых прыгунов по мере прохождения игры;
* рейтинг игроков (в рейтинге будут учитываться результаты самого игрока с разных прохождений. "Борьба с самим собой");
* изменение пространства вокруг прыгуна по мере прохождения препятствий;
* "опасные" платформы для прыгуна
